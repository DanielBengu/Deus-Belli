Events are stored in the Assets\Resources\Scripts\Rogue\Encounters.TXT file, each line represents a single event and every attribute is separated with the ; character, following this scheme:
Image name (as its stored); Title of Event; Description of Event; Choice 1 text; Choice 1 function associated; Choice 2 text; Choice 2 function associated;

Functions:
Functions are structured as follows:
g123
First char indicates the type of function and the rest of the string indicates the reward data (amount of gold, type of unit gifted).

If an event has multiple rewards they are split with a , to read them separately, e.g.:
g123,u456,e789
